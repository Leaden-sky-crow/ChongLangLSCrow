## 技术说明

### 两行文本显示

使用 `div` 包裹两个 `p` 标签：
```tsx
<div className="text-xs text-muted-foreground">
  <p>第一行文本</p>
  <p>第二行文本</p>
</div>
```

**优点**：
- 语义清晰
- 自动换行
- 保持样式一致
- 易于维护

### useEffect 钩子使用

**组件挂载时执行**：
```typescript
useEffect(() => {
  loadNotifications()
}, []) // 空依赖数组，只在挂载时执行一次
```

**依赖项变化时执行**：
```typescript
useEffect(() => {
  if (open) {
    loadNotifications()
  }
}, [open]) // open 变化时执行
```

**两个 useEffect 配合使用**：
- 第一个：组件挂载时加载一次
- 第二个：菜单打开时刷新
- 不会造成重复请求，因为依赖不同

### 通知加载逻辑

```typescript
// 1. 组件挂载时加载
useEffect(() => {
  loadNotifications()
}, [])

// 2. 菜单打开时刷新
useEffect(() => {
  if (open) {
    loadNotifications()
  }
}, [open])
```

**执行流程**：
1. 组件首次渲染 → 执行第一个 useEffect → 加载通知
2. 用户打开菜单 → 执行第二个 useEffect → 刷新通知
3. 用户关闭菜单 → 无操作
4. 用户再次打开菜单 → 执行第二个 useEffect → 刷新通知
